# Обрезка ссылок с помощью Битли
### Как установить
Python3 должен быть уже установлен.
Затем используете 'pip' (или 'pip3' , если есть конфликт с Python2) для установки зависимостей:
'''
pip install - r requirements.txt
'''
### Цель проекта
